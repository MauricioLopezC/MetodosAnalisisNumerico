# Arranque
para iniciar el servidor:
```shell
python3 server.py
```

para poner en marcha el servidor en modo debug:
```bash
flask --app server run --debug --host=0.0.0.0
```

## lista de dependencias installar con pip
- scipy
- flask
- flask-cors

Instalar con: 
```bash
pip install flask
pip install -U flask-cors
pip install scipy
```
